# Tutorial Example Files

These files are used by `docs/tutorials/*`.

- `first_program.qsol`
- `first_program.instance.json`
- `assignment_balance.qsol`
- `assignment_balance.instance.json`

Try:

```bash
uv run qsol run examples/tutorials/first_program.qsol --instance examples/tutorials/first_program.instance.json --out outdir/first_program --sampler exact
```
